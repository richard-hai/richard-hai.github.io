Spotify Music Player

A custom made UI to play any song in Spotify's database.

Demo

This project was specifically created to tackle learning how to use HTML5's multimedia audio element and using Spotify's oAuth 2.0 authentication process. The demo link actually takes you to a backend express server via the express /login route that will request an authorization token from spotify with the client ID which will in turn send you back to to the express server via the express /callback route designated in the Spotify developer settings. This verified request token received from spotify will then be used alongside our client secret id to request the needed authorization token which we receive in a callback. Once we receive the temporary token, the backend express server will re-route the user to the actual front-end https://richard-spotify-music-player.herokuapp.com/ with the token in the URL. Finally, our app will be able to make any fetch request for JSON data to have full functionality in the app.

Tech - Javascript, HTML5/ CSS3, React.js, React Bootstrap, Node.js/ Express.js, Spotify's API, Heroku
